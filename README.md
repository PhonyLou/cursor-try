# Cursor Try

这是一个使用 Spring Boot 构建的 RESTful API 示例项目。

## 功能特性

- 基于 Spring Boot 3.x
- 提供 RESTful API 接口
- 支持 Docker 容器化部署
- 使用轻量级基础镜像

## 快速开始

### 使用 Docker Compose 运行

```bash
docker compose up -d
```

应用将在 http://localhost:8080 上运行。

### API 端点

- GET /api/hello - 返回欢迎消息

## 技术栈

- Java 17
- Spring Boot 3.2.3
- Gradle 8.5
- Docker
